# 朋友在线德州扑克

一个零依赖的本地 Web 项目，用 Node.js 内置 HTTP 服务和原生 WebSocket 实现好友房德州扑克。

## 运行

```bash
npm start
```

如果本机没有全局 Node.js，在 Codex 桌面环境里可以用：

```bash
/Users/yq225/.cache/codex-runtimes/codex-primary-runtime/dependencies/node/bin/node server.js
```

打开：

```text
http://127.0.0.1:3000
```

## 怎么和朋友玩

1. 打开页面后输入昵称入座。
2. 房主可以在大厅或摊牌后设置小盲、大盲、默认买入和买入上限。
3. 点击“复制邀请链接”，把带 `room` 参数的链接发给朋友。
4. 玩家可以在大厅或摊牌后买入/补码。
5. 房主在至少 2 人入座后点击“开始/下一手”。
6. 轮到自己时可以弃牌、过牌、跟注、加注，或用快捷按钮选择最小加注、半池、底池、All in。

## 已实现

- 房间链接和最多 6 人入座
- WebSocket 实时同步
- 私有手牌、公共牌、底池和下注状态
- 小盲/大盲、翻牌前、翻牌、转牌、河牌、摊牌
- 房主牌桌设置：盲注、默认买入、买入上限
- 大厅/摊牌后的买入与补码
- 快捷下注和 All in
- 基础德州牌型评估、主池和边池结算
- 断线后用同一浏览器回到座位
- 桌面和移动端响应式布局

## 当前边界

- 房间状态保存在服务器内存中，重启后会清空。
- 适合朋友局原型；正式真钱或严肃竞技需要补充鉴权、防作弊、日志审计和完整边池规则。
