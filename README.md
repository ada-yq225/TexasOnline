# 朋友在线德州扑克 🃏

一个轻量级的在线德州扑克游戏，适合朋友间开房对局。无需注册、无需下载，打开链接即可玩。

---

## 快速开始

### 本地运行

```bash
npm start
```

浏览器打开 `http://127.0.0.1:3000` 即可。

### 和朋友远程玩

你需要将服务暴露到公网。推荐使用内网穿透工具（如 serveo、ngrok）：

```bash
# 以 serveo 为例
ssh -R 80:localhost:3000 serveo.net
```

服务启动后会返回一个公网链接，发给朋友就能一起玩了。

## 怎么玩

### 1. 进入游戏

打开页面后输入你的昵称，点击「创建新房间」或选择已有房间加入。

### 2. 入座与买入

进入房间后，点击「加入牌桌」选择一个空位坐下。默认买入金额会自动补充到你的筹码中。

### 3. 设置牌桌（房主）

房主可以在大厅或摊牌后修改牌桌参数：

- **小盲 / 大盲**：盲注金额，大盲必须是小盲的 2 倍
- **默认买入**：玩家入座时自动获得的筹码
- **买入上限**：玩家补码时不能超过此金额

### 4. 开始游戏

至少 2 人入座后，房主点击「开始」按钮即可发牌。每手牌结束后房主点击「下一手」继续。

### 5. 你的回合

轮到你行动时，座位会高亮显示，你可以：

| 操作 | 说明 |
|------|------|
| **弃牌** | 放弃本手牌，已投入的筹码不会退还 |
| **过牌** | 当前无人加注时可选择过牌 |
| **跟注** | 支付与当前最高下注相同的筹码 |
| **加注** | 输入加注金额或使用快捷按钮 |
| **All in** | 押上全部筹码 |

**快捷下注按钮**：最小加注 · 半池 · 底池 · All in，点击后自动填入加注框。

### 6. 游戏流程

`翻牌前 → 翻牌 → 转牌 → 河牌 → 摊牌结算`

每轮依次行动，直到所有人下注平齐或全部弃牌。最后摊牌比牌，最佳牌型获胜。

### 7. 补码

在大厅或摊牌结算后，你可以通过侧边栏的「买入/补码」补充筹码。

### 8. 重新开始

房主可以随时点击右上角的「重新开始」按钮，将牌桌恢复到初始状态（所有人重新买入）。

## 玩家统计

每位玩家的座位上会显示三项数据：

- 🏆 **胜率** — 获胜手数 ÷ 参与手数
- 💰 **入池率 (VPIP)** — 主动入池次数 ÷ 参与手数
- 📈 **总收益** — 累计盈亏（绿色正、红色负）

## 其他功能

### 聊天
侧边栏内置聊天面板，支持实时文字交流。

### 牌局记录
侧边栏底部的牌局记录显示最近 12 条游戏事件（发牌、下注、加注、获胜等）。

### 音效
游戏内置音效（发牌、下注、All in、获胜等），可通过右上角 🔊 按钮开关。

### 观战
座位满或游戏进行中加入会自动进入观战模式，可以看到公共牌和玩家行动。

### 断线重连
关闭页面或断线后，用同一浏览器重新打开链接即可回到原位。

### 复制邀请链接
点击顶栏的「复制邀请链接」即可获取带房间号的链接，发给朋友直接加入。

## 牌型大小（从大到小）

| 牌型 | 说明 |
|------|------|
| 皇家同花顺 | A-K-Q-J-10 同花 |
| 同花顺 | 五张连续同花色 |
| 四条 | 四张相同点数 |
| 葫芦 | 三条 + 一对 |
| 同花 | 五张同花色 |
| 顺子 | 五张连续（A 可作最小） |
| 三条 | 三张相同点数 |
| 两对 | 两个对子 |
| 一对 | 一个对子 |
| 高牌 | 比最大单张 |

## 技术说明

- 零外部依赖，基于 Node.js 内置 HTTP 模块和原生 WebSocket
- 房间状态存储在服务器内存中，服务重启后数据清空
- 最大支持 6 人同时游戏 + 观战
- 适合朋友娱乐局，非真钱博彩用途
